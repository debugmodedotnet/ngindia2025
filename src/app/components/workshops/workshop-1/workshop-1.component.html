Keeping up with Angular's rapid pace is already challenging. What’s even harder is ensuring your application remains stable and evolvable while integrating upcoming updates and features.

This workshop is for advanced Angular developers who want to:

  • Build an evolvable architecture that scales with their code.
  • Leverage the latest Angular features: Signals, resource, linkedSignal, zoneless execution, and incremental hydration.
  • EManage state efficiently using Signals, Angular Query, and the NgRx SignalStore for complex scenarios.
  • Protect their applications with modern testing using Playwright Component Testing.

This workshop is the full package for your Angular journey.

Agenda

Introduction: Angular's Latest Features
  • Simple reactivity with Signals.
  • Optimized performance through zoneless execution.
  • Enhanced rendering with incremental hydration.
  • Simplified workflows with Single-File Components and beyond.
  • Understand why these features exist and how they improve your development experience.
Session 1: Evolvable and Scalable Architecture
  • The challenge of achieving a stable and evolvable architecture.
  • Splitting applications into feature modules and domains.
  • Ensuring architectural boundaries with Sheriff.
  • Scaling applications with Native Federation for enterprise-grade modularity.
Session 2: Modern State Management with Signals
  • Managing state with Angular's native Signals and tools like linkedSignal and (rx)Resource.
  • Keeping server and client state in sync with Angular Query.
  • Handling complex state requirements with the NgRx SignalStore.
  • Best practices for building clear, efficient, and maintainable state management.
Session 3: Ensuring Stability with Testing & CI
  • Modern testing strategies with Playwright Component Testing for Angular.
  • Writing reliable, efficient component tests to ensure stability.
  • Integrating tests into your CI pipeline to maintain production readiness.
Closing
  • Recap of key concepts and takeaways.
  • Open Q&A: Address advanced challenges and real-world use cases.
  • Final thoughts on building sustainable, scalable, and future-proof Angular applications.

What You Will Learn

By the end of this workshop, participants will:

  • Master evolvable architecture concepts for long-term scalability and stability.
  • Use Angular’s latest tools and features to build modern applications.
  • Efficiently manage state with Signals, SignalStore, and Angular Query.
  • Write modern, reliable tests using Playwright Component Testing.
  • Apply incremental hydration, zoneless execution, and Single-File Components for optimized performance.

About the Trainer

Rainer Hahnekamp Trainer and Consultant @AngularArchitects.io

Rainer Hahnekamp is a Google Developer Expert in Angular and a prominent figure in the Angular community. He is:

  • A member of the NgRx Team.
  • The lead developer behind Sheriff, a tool for enforcing architectural boundaries in TypeScript projects.
  • A contributor to widely used libraries like Native Federation and ngrx-toolkit.
  • Currently part of a team bringing Playwright Component Testing to Angular.

Rainer is actively involved in community projects, sharing his knowledge through workshops at Angular Architects, conference talks, and the ng-news series. His deep experience in enterprise Angular applications and cutting-edge tooling ensures participants receive both theoretical knowledge and hands-on expertise for building modern, scalable applications.
